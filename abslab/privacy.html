プライバシーポリシー

1第1条（個人情報の定義）

「個人情報」とは、個人情報保護法にいう「個人情報」を指すものとし、生存する個人に関する情報であって、当該情報に含まれる氏名、生年月日、住所、電話番号、連絡先その他の記述等により特定の個人を識別できる情報、及び容貌、指紋、声紋にかかるデータ、健康保険証の保険者番号などの当該情報単体から特定の個人を識別できる情報（個人識別情報）を指します。

2第2条（個人情報の収集について）

本アプリは、開発者（以下「当方」）のサーバーにユーザーの個人情報を送信・収集することは一切ありません。

本アプリで扱う以下の情報は、すべてユーザーのデバイス上、またはユーザー自身のApple iCloudアカウント内にのみ保存されます。当方がこれらのデータを閲覧・取得・利用することはできません。

  • 健康・フィットネスデータ（体重、食事記録、運動記録、水分摂取量、睡眠時間）：デバイス上およびiCloudにのみ保存されます
  • HealthKitデータ（体重、歩数、消費カロリー）：Apple HealthKitとの連携により読み取り・書き込みを行います（ユーザーの許可が必要です）。HealthKitデータはAppleの管理下にあり、当方はアクセスできません
  • プロフィール情報（ニックネーム、目標体重、年齢、性別等）：ダイエット目標設定のために使用し、デバイス上およびiCloudにのみ保存されます
  • 写真（ビフォーアフター比較用の体型写真）：ユーザーが任意で撮影・保存するもので、デバイス上およびiCloudにのみ保存され、外部に送信されることはありません

3第3条（情報の利用目的）

前条の情報は、すべてユーザーのデバイス上で以下の目的にのみ使用されます。当方のサーバーへの送信は行いません。

  • カロリー・PFCバランスの計算と表示
  • AIコーチによるパーソナルアドバイスの生成（デバイス上で処理）
  • 体重トレンドの分析と統計予測
  • 運動記録の管理と習慣化サポート
  • ユーザー自身のデバイス間でのデータ同期（ユーザーのiCloudアカウント経由）

4第4条（利用目的の変更）

利用目的を変更した場合には、変更後の目的について本プライバシーポリシーを改定し、本ページ上にて公表するものとします。

5第5条（データの保存場所）

  • デバイス上：すべてのデータはユーザーのデバイスにローカル保存されます
  • iCloud：Apple CloudKit を使用して、ユーザー自身の複数デバイス間でデータを同期します。データはAppleのiCloudサーバーに暗号化して保存されます

当方は独自のサーバーを持たず、ユーザーのデータを収集・保存・閲覧する手段を一切持ちません。iCloud上のデータもユーザー個人のアカウントに紐づいており、当方がアクセスすることはできません。

6第6条（個人情報の第三者提供）

当方は、ユーザーの個人情報を第三者に販売、貸与、共有、または提供することは一切ありません。

当方はユーザーのデータを収集・保有していないため、第三者に提供すること自体が不可能です。ただし、以下の場合はこの限りではありません。

  • 法令に基づく場合
  • 人の生命、身体または財産の保護のために必要がある場合であって、本人の同意を得ることが困難であるとき
  • 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合

7第7条（広告・トラッキング）

本アプリは、広告を表示しません。また、ユーザーの行動を追跡するトラッキングツールやアナリティクスツールを使用していません。

8第8条（HealthKit データの取り扱い）

本アプリはApple HealthKitと連携します。HealthKitから取得したデータについて、Apple の HealthKit ガイドラインに従い、以下を遵守します。

  • HealthKitデータは、本アプリの機能提供のためにのみ、デバイス上で使用します
  • HealthKitデータを広告やマーケティングに使用することはありません
  • HealthKitデータを第三者に提供・販売することはありません
  • HealthKitデータを当方のサーバーや外部サーバーに送信することはありません

9第9条（個人情報の開示・訂正・削除）

すべてのデータはユーザーが完全に管理しています。

  • アプリ内の設定からデータを確認・訂正・削除できます
  • アプリをデバイスから削除することで、ローカルデータは完全に削除されます
  • iCloud上のデータは、端末の設定 → Apple ID → iCloud から削除できます

当方はユーザーのデータを保有していないため、当方への開示請求・削除依頼は不要です。

10第10条（アプリ内課金・サブスクリプション）

本アプリでは、Appleのアプリ内課金システム（In-App Purchase）を通じてサブスクリプション（定期購読）を提供しています。課金に関する事項はすべてApple Inc.のシステムおよび規約に準拠します。

  • 課金方法：すべての支払いはAppleのApp Store経由で処理されます。当方が直接クレジットカード情報等の決済情報を取得・保管することはありません
  • 自動更新：サブスクリプションは、現在の期間が終了する24時間前までに自動更新がオフにされない限り、自動的に更新されます
  • 解約方法：サブスクリプションの解約は、端末の設定 → Apple ID → サブスクリプション から行うことができます。解約後も、支払い済みの期間が終了するまでプレミアム機能をご利用いただけます
  • 無料トライアル：無料トライアル期間が提供される場合、トライアル終了の少なくとも24時間前までに解約しなければ、自動的に有料サブスクリプションに移行します
  • 返金：課金・返金に関するお手続きはすべてAppleが管理しています。返金をご希望の場合は、Appleサポート（reportaproblem.apple.com）よりAppleに直接お問い合わせください。当方では返金処理を行うことができません

11第11条（プライバシーポリシーの変更）

本プライバシーポリシーの内容は、法令その他本ポリシーに別段の定めのある事項を除いて、ユーザーに通知することなく変更することができるものとします。

変更後のプライバシーポリシーは、本ページに掲載したときから効力を生じるものとします。重要な変更がある場合は、アプリ内でもお知らせします。
