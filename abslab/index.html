よくあるご質問

  • データが同期されません
    iCloudがオンになっているかご確認ください。設定 → Apple ID → iCloud で確認できます。
  • 食品が見つかりません
    手入力機能で自由に食品を追加できます。食事画面の「手入力」タブをご利用ください。
  • サブスクリプションの解約方法
    設定 → Apple ID → サブスクリプション から管理できます。

アプリについて

「腹筋を割ろう」は、2,847件の科学論文を分析して作られた本格的なダイエット・ボディメイク管理アプリです。

  • 23,000品目以上の食品データベース
  • PFC（タンパク質・脂質・炭水化物）バランスの自動管理
  • 6人のAIコーチによるパーソナルアドバイス
  • HIIT・筋トレ・有酸素運動の記録と習慣化
  • 移動平均と統計予測による体重管理
