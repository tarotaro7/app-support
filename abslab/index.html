よくあるご質問

  • データが同期されません
    iCloudがオンになっているかご確認ください。設定 → Apple ID → iCloud で確認できます。
  • 食品が見つかりません
    手入力機能で自由に食品を追加できます。食事画面の「手入力」タブをご利用ください。
  • サブスクリプションの解約方法
    設定 → Apple ID → サブスクリプション から管理できます。
